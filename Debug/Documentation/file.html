Index>Opening, Saving, and Creating Documents


To save your document, go to File>Save. Everytime when you visit file, you save your document and reload it. Note: you must have have opened or created a file and overwites whatever document you opened


To create a new document, and shift the file you are working with, go to File>SaveAs. This creates a new file, and writes the rtf to it.


To open a document, go to File>Open. If the document isn't in a valid rich text format, the document will load its raw contents. Note, saving the document rewrites all contents with the rtf text!
