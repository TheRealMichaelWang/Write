To go to the Vocabulary Executor, go to Vocabulary>. Vocabulary will write the definition one line below the word(Write will parse a whole line as one word). Vocabulary will also create one line in between. To use certain vocab sets, load in special dictionaries. Note: the spelling is crucial to defining the word.
To edit the dictionary manualy, simply add a { between term and definition and |between different terms.
